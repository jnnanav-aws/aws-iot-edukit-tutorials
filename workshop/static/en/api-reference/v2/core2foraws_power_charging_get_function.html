Get the current charging status of the built-in battery from the power management unit (PMU).

Get the current charging status of the built-in battery from the power management unit (PMU).

Note
A fully charged battery (~4.2v) will not receive further charge.
Parameters
[out]statusPointer to the boolean value of the charging status. 1 (true) if it is currently charging, 0 (false) if it is not charging or fully charged.
Returns
esp_err_t.
  • ESP_OK : Success
  • ESP_ERR_INVALID_ARG : Driver parameter error
